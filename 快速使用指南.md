# 💹 金融产品价格监控系统 - 快速使用指南

## 🚀 最简单的使用方式（推荐）

### 下载即用 - 无需安装任何软件！

1. **下载文件**
   - 在 GitHub 仓库中找到 `FinancialMonitor.html` 文件
   - 点击下载到你的电脑

2. **双击运行**
   - 找到下载的 `FinancialMonitor.html` 文件
   - 双击打开，会在浏览器中自动运行
   - 支持所有现代浏览器：Chrome、Safari、Firefox、Edge

3. **开始使用**
   - 点击"添加资产"标签
   - 输入股票代码（如 AAPL）、ETF代码（如 SPY）或数字货币名称（如 bitcoin）
   - 点击"价格监控"查看实时价格和涨幅概率

## ✨ 主要功能

### 📊 实时价格监控
- 监控股票、ETF、数字货币的实时价格
- 显示24小时涨跌幅
- 自动刷新价格数据

### 🎯 概率分析
- 计算每个资产在6个月内涨30%以上的概率
- 基于历史波动率的蒙特卡洛模拟
- 年化波动率指标

### 🔔 价格提醒
- 设置价格达到指定值时的提醒
- 支持"高于"和"低于"两种条件
- 邮箱通知功能

### 💾 数据管理
- 所有数据存储在浏览器本地
- 支持导出/导入数据备份
- 一键清除所有数据

## 📖 使用示例

### 1. 添加股票监控

**股票代码示例：**
- AAPL - Apple Inc.
- MSFT - Microsoft
- TSLA - Tesla
- GOOGL - Google
- AMZN - Amazon
- NVDA - NVIDIA

**操作步骤：**
1. 点击"➕ 添加资产"标签
2. 选择"股票 (Stock)"
3. 输入代码"AAPL"
4. 点击"添加资产"

### 2. 添加ETF监控

**ETF代码示例：**
- SPY - S&P 500 ETF
- QQQ - Nasdaq 100 ETF
- VOO - Vanguard S&P 500 ETF
- IWM - Russell 2000 ETF
- DIA - Dow Jones ETF

**操作步骤：**
1. 选择"ETF"
2. 输入代码"SPY"
3. 点击"添加资产"

### 3. 添加数字货币监控

**数字货币示例：**
- bitcoin - 比特币
- ethereum - 以太坊
- cardano - 艾达币
- solana - 索拉纳
- polkadot - 波卡

**注意：** 数字货币需要使用全称小写，不是简写！
- ✅ 正确：bitcoin
- ❌ 错误：BTC

### 4. 设置价格提醒

1. 在资产卡片上点击"设置提醒"按钮
2. 输入你的邮箱地址
3. 选择条件（价格高于/低于）
4. 输入目标价格
5. 点击"创建提醒"

当价格达到你设定的值时，系统会在页面上显示通知。

## ⚙️ 配置API密钥（可选）

### Alpha Vantage API（用于股票/ETF）

**为什么需要：**
- 默认使用演示模式，每天只能查询5次
- 获取免费API密钥后可以每分钟查询5次

**获取步骤：**
1. 访问：https://www.alphavantage.co/support/#api-key
2. 填写姓名和邮箱
3. 点击"GET FREE API KEY"
4. 复制你的API密钥

**配置步骤：**
1. 点击"⚙️ 设置"标签
2. 在"Alpha Vantage API Key"框中粘贴你的密钥
3. 点击"保存设置"

### CoinGecko API（用于数字货币）

数字货币数据使用CoinGecko的免费API，无需注册即可使用！

## 🔧 高级功能

### 自动刷新设置

1. 点击"⚙️ 设置"标签
2. 调整"自动刷新间隔"（1-60分钟）
3. 点击"保存设置"

系统会按照设定的时间间隔自动刷新所有资产价格。

### 数据导出和导入

**导出数据：**
1. 点击"⚙️ 设置"标签
2. 点击"导出数据"按钮
3. 保存JSON文件到你的电脑

**导入数据：**
1. 点击"导入数据"按钮
2. 选择之前导出的JSON文件
3. 所有资产和提醒会被恢复

**用途：**
- 在不同电脑间同步数据
- 定期备份你的监控列表
- 更换浏览器时迁移数据

## 💡 使用技巧

### 1. 建立多元化投资组合监控

添加不同类型的资产以分散风险：
- 3-5支蓝筹股（如 AAPL、MSFT、GOOGL）
- 2-3个ETF（如 SPY、QQQ）
- 1-2个数字货币（如 bitcoin、ethereum）

### 2. 设置合理的价格提醒

**上涨提醒：**
- 设置目标利润点（如买入价+20%）
- 考虑设置止盈点

**下跌提醒：**
- 设置止损点（如买入价-10%）
- 及时止损避免更大损失

### 3. 关注概率指标

**高概率（>50%）：**
- 绿色显示
- 表示历史数据显示该资产有较大可能上涨30%

**中等概率（25%-50%）：**
- 橙色显示
- 需要更谨慎地分析

**低概率（<25%）：**
- 红色显示
- 可能需要重新评估投资策略

**注意：** 概率仅基于历史数据，不构成投资建议！

### 4. 定期更新数据

- 点击资产卡片上的"刷新"按钮更新单个资产
- 系统会自动每5分钟刷新所有资产（可在设置中调整）
- 建议至少累积10天以上的历史数据以获得更准确的概率计算

## ❓ 常见问题

### Q1: 为什么某些资产价格显示为0或加载失败？

**可能原因：**
- 股票/ETF代码输入错误
- 数字货币使用了简写而非全称
- API请求次数超限（使用演示密钥）
- 网络连接问题

**解决方法：**
- 检查代码拼写
- 数字货币使用全称小写（如 bitcoin 而非 BTC）
- 配置自己的Alpha Vantage API密钥
- 等待几分钟后刷新

### Q2: 概率显示为0%是什么意思？

**原因：**
- 该资产刚添加，历史数据不足（需要至少10个数据点）

**解决方法：**
- 多次刷新该资产以累积历史数据
- 等待自动刷新功能收集更多数据
- 通常需要1-2天才能获得较准确的概率

### Q3: 价格提醒会发送真实邮件吗？

**当前版本：**
- 提醒触发时会在页面上显示通知
- 不会发送真实邮件（需要后端服务器支持）

**如果需要邮件功能：**
- 可以使用完整版应用（包含Node.js后端）
- 或等待后续版本更新

### Q4: 数据会丢失吗？

**数据存储：**
- 所有数据存储在浏览器的localStorage中
- 只要不清除浏览器缓存，数据会一直保留

**建议：**
- 定期导出数据备份
- 重装系统或清除浏览器前记得导出

### Q5: 可以在手机上使用吗？

**完全可以！**
- 支持iOS Safari和Android Chrome
- 响应式设计，自动适配手机屏幕
- 使用方法与电脑完全相同

### Q6: API密钥安全吗？

**隐私保护：**
- API密钥存储在你的浏览器本地
- 不会上传到任何服务器
- 只有你自己可以访问
